Books

There are many, many books on Haskell. This course will be academically driven, so I'll only recommend academically driven books for this course. However, there are other books that are more suitable to using Haskell in the "Real World".

  • Programming Haskell by Graham Hutton
  • The Haskell Road to Logic, Maths and Programming by Jan van Eijck
  • Haskell: The craft of functional programming by Simon Thompson

The first one is an excellent book, the second is also excellent, but more difficult to read, and the third and final book is easier to read, but not (in my opinion) as high a standard as the first two.
